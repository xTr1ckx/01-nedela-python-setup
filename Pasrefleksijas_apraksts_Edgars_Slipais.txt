Ir pagājis kāds laiks kopš es pabeidzu Bakalaura grādu Datorspēļu izstrādes kursā, tāpēc man aizņēma nedaudz laika ieiet atpakaļ ritmā saistībā ar GitHub uzstādi un kodēšanu. Tomēr iepriekš iegūtās zināšanas par GitHub un Visual Studio Code ļoti palīdzēja šim procesam noritēt plūstoši un man personīgi nebija lielas aizķeršanās. Dažu nosacījumu formulējums pirmā mājasdarba failā bija nedaudz mulsinoši, toties es ticu, ka visu esmu izdarījis pareizi.
